자바 실행

1. 컴파일 단계 : 사람이 사용한 문자를컴퓨ㅓ가 알 수 있도록 0과 1로 번역
	필요 명령어 : javac
2. 인터프리터 : 0과 1로된 바이너리 코드를 해석하고 실행하는 단계
	필요 명령어 : java
3. 실행하고자 하는 파일의 위치를 저장
4. cmd 창을 이용해서 해당 위치로 이동한다.
	cd 파일 위치
	위치 확인은 dir 명령어로 확인
 컴파일 실행 : javac 파일이름.java
 인터프리터 실행 : java 파일이름

명명법(식별자) : 이름 만드는 방법

1. 클래스 : 첫글자는 대문자. 나머지는 소문자
 	보통 명사를 사용., 숫자는 중간이나 끝에 사용가능
	단어가 두 단어 이상일 때는 단어의 첫글자는 무조건 대문자 사용
	객체를 생성하기 위한 자원은 클래스이다.
	객체 ; 사용할 수 있는 자원
	파일로 저장할 수 있는 단위(클래스 이름이 저장 이름)
	자바  API 는 클래스들로 이루어져 있다.
2. 메소드 ; 클래스 안에 존재
	메소드이름 () 형태이다
	동작, 기능을 의미
	첫글자는 소문자
	두 단어 이상일 때 두 번째 단어의 첫글자는 대문자
	javaTestExam
3. 변수와 상수 : 클래스나 메소드 안에 존재
	1) 변수 : 첫 글자는 소문자를 사용
		두 단어 이상일 때 두 번째 단어부터 첫글자는 대문자
		(명명법은 메소드와 같다)
	2) 상수 : 모든 글자가 대문자.
		단어와 단어 사이에 under score(_)를 사용
		(JAVA_HOME)


	변수와 상수
	변수 : 프로그램의 시작에서 끝낼때까지 언제든지 가지고 있는 데이터가 변할 수 있는 저장 공간을 의미
	상수 : 프로그램의 시작에서 끝낼때까지 한번 지정한 데이터는 변할 수 없는 저장 공간을 의미

	자바에서 변수와 상수를 표현하는 방법
	변수  : 
		자료형 변수이름; //변수 선언이라 함 ( 한번만 하면 된다)
		변수이름 = 데이터; //변수에 데이터를 저장 (절대로 같다라는 의미 아니다)
		자료형 변수 이름 = 데이터;

	상수 : static final 자료형 상수이름;
		final 자료형 상수이름;
